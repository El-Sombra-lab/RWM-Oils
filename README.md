# RWM Oils Petroleum Group

This is the readme file for the code provided. The code represents the HTML structure and styling for the official website of RWM Oils Petroleum Group. 

## Description
The code consists of an HTML document that creates a website for RWM Oils Petroleum Group. It includes various sections such as a navigation bar, cover section, about section, products section, schedule section, contact section, and footer. The code also includes CSS styles and JavaScript scripts to enhance the functionality and appearance of the website.

## Files
- `index.html`: The main HTML file that represents the structure of the website.
- `aquamarine.css`: CSS file for custom styles used in the website.
- `navbar-ontop.js`: JavaScript file for making the navbar transparent when scrolled to the top.
- `animate-in.js`: JavaScript file for adding animated entrance effects to elements.

## Usage
To use this code, follow these steps:
1. Download all the files mentioned above and place them in the same directory.
2. Open the `index.html` file in a web browser.

## Dependencies
The code relies on the following dependencies:
- `aquamarine.css`: Custom CSS file for styling.
- Font Awesome (version 4.7.0): CSS file for adding icons to the website.
- `navbar-ontop.js`: JavaScript file for the transparent navbar functionality.
- `animate-in.js`: JavaScript file for animated entrance effects.

Make sure that the dependencies are correctly linked in the HTML file for the code to work properly.
